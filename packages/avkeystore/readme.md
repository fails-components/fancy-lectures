# Fancy automated internet lecture system (**FAILS**) - components

This package is part of FAILS.
A web-based system developed out of university lectures.

This package is part of the audio, video stack of **FAILS**,
which may be useful outside of fails itself.

It provides a simple keystore class, that used for storing and updating
the keys used for E2E encryption.

Feedback on errors/issues is appreciated via github's issue function.
