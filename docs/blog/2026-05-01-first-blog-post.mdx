Preparing for the 2.0 release of FAILS, FAILS is getting a documentation.
Though most parts of the documentation is generated with the help of AI,
the endover will facilitate setting up FAILS at other sites.

The documentation will be installed along side every FAILS installation starting
with 2.0.
