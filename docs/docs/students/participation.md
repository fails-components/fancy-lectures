# Real-Time Participation: Polls and Communication

FAILS provides multiple tools for students to engage with the material, ask questions, and test their knowledge anonymously during a live lecture.

## 🗳️ Anonymous Polling System
Polls allow students to provide immediate feedback on lecture topics without fear of judgment.

### How Voting Works
*   **Anonymity:** All polls conducted via FAILS are completely anonymous; the teacher cannot identify which answer was selected by which student.
*   **Voting Mechanics:** Students select their choice(s) based on whether a poll allows for **single selection** or **multiple selections**.
*   **Verification:** While your vote remains private, each student receives a unique **ballot ID**. This ID can be used later to verify that the vote was successfully counted.

## Chat Question Functionality
Students can submit questions directly onto the main blackboard display using the chat feature:

1.  **Posting:** Click the **chat button** and type your question into the message box.
2.  **Formatting Support:** You can include complex mathematical notation using **LaTeX math** within your question.
3.  **Privacy:** Your name is automatically truncated on the main blackboard display, maintaining a level of privacy suitable for large class sizes.

### 🔊 Hybrid Q&A (Audio/Video Questions)
For situations requiring direct spoken or visual interaction:
*   **Initiating a Question:** Press the dedicated **question button**. Select that you wish to pose an "audio-video question" and send it.
*   **Live Interaction:** The lecturer has control over whether they grant or deny permission for the student to speak or broadcast video content during the lecture.
