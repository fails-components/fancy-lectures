# Developer's guide
FAILS component consists of several reusuable opensource components.
Most prominent example is the webtransport package for node.js `@fails-components/webtransport`.
However, also the blackboard and the packages for audio/video streaming are in principle reusuable.
If time permits, or if AI can autogenerate it, a detailed described will be added.
So far, look for example in the `packages` directory of the main repo.
